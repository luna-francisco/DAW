# DAW | Desarrollo de Aplicaciones Web

Repositorio personal del ciclo de **Desarrollo de Aplicaciones Web (DAW)**.

Aquí organizo el trabajo del curso por **asignaturas**, de forma que cada carpeta reúne el material generado durante mi formación: **apuntes, ejercicios prácticos y proyectos desarrollados por mí**.

## Qué encontrarás

- **Estructura por asignaturas** para localizar rápido cada bloque de trabajo.
- **Apuntes y material de apoyo** usados durante el curso.
- **Prácticas y proyectos** que reflejan mi evolución técnica y mi forma de trabajar.

## Organización del proyecto

Cada carpeta principal corresponde a una asignatura del ciclo:

- `BBDD`
- `Entornos`
- `LenguajeMarcas`
- `Programacion`
- `Sistemas`

Dentro de cada una se pueden encontrar:

- Apuntes y documentación
- Ejercicios de clase
- Prácticas individuales
- Proyectos desarrollados durante el curso

## Enfoque

Este repositorio funciona como una **base de trabajo y aprendizaje real**. No es una colección aislada de ejemplos, sino una muestra ordenada de cómo he ido aplicando conceptos de desarrollo web, programación y sistemas a lo largo del curso.

## Visión general

Este repositorio permite ver de forma rápida:

- Cómo organizo mi trabajo técnico
- Qué tecnologías y materias he ido tocando
- Ejemplos prácticos de ejercicios y proyectos desarrollados durante mi formación
